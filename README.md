# Restaurant Website (Frontend)

This is a frontend project for a restaurant website. It focuses on design, layout, and user experience without any backend functionality. The site includes a responsive design, interactive menu, reservation form, and smooth navigation to create a welcoming digital dining experience.

## Features
- Responsive layout for mobile, tablet, and desktop
- Interactive food menu
- Reservation form (UI only, no backend)
- Clean and modern user interface

## Tech Stack
- HTML
- CSS
- JavaScript
